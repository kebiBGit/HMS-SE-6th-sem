
# Himalayan Haven Hotel Website

This website was designed and developed by myself along with two other friends as a collaborative project. All code, design, and content were created by us from scratch.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

To run the project locally:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Deployment

You can deploy this project to any static hosting platform that supports Vite/React builds (e.g., Vercel, Netlify, GitHub Pages).

## Custom Domain

To use a custom domain, follow your hosting provider's instructions for domain setup.
